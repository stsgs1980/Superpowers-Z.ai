# Help

Please explain the following to the user:

## Available Commands

### /commit
Create a git commit with a meaningful message.

### /code-review
Perform a code review of current changes.

### /feature-dev {argument}
Develop a new feature following a structured workflow.

### /hookify {argument}
Create a hookify rule from natural language.

### /review-pr {argument}
Review a pull request with multiple perspectives.

### /clean-gone
Clean up stale local branches deleted from remote.

### /commit-push-pr
Commit, push, and open a pull request.

### /configure
Enable or disable hookify rules interactively.

### /create-plugin {argument}
Guided plugin creation workflow.

### /revise-claude-md
Update CLAUDE.md with learnings from this session.

- **systematic-debugging**: 4-phase debugging with Iron Law
- **verification-before-completion**: No false claims
- **brainstorming**: Design-first workflow
